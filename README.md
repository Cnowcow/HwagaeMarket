  ###### 4. jQuery
  ###### 5. Ajax
  ###### 6. Css 3
  ###### 7. SpringBoot
  ###### 8. JPA
  ###### 9. MySql DB
  ###### 10. GitHub
  ###### 11. Gradle
  ###### 12. InteliJ
  ###### 13. Visual Studio
  ###### 14. BootStrap
